# VS Code で GitHub Copilot を使う

> 前提：GitHub Education の承認が完了していること（Copilot が無料で利用可能）

---

## 1. VS Code のインストール

### まだインストールしていない場合

1. [https://code.visualstudio.com](https://code.visualstudio.com) にアクセス
2. **「Download for Windows」**（または macOS / Linux）をクリック
3. ダウンロードしたインストーラーを実行
4. インストール時のオプション：
   - **「PATHへの追加」**：チェックを入れる（推奨）
   - **「エクスプローラーのコンテキストメニューに追加」**：チェックを入れる（推奨）
5. インストール完了後、VS Code を起動

---

## 2. GitHub Copilot 拡張機能のインストール

1. VS Code を起動
2. 左のサイドバーにある **拡張機能アイコン**（四角が4つのマーク）をクリック
   - またはキーボードショートカット：`Ctrl + Shift + X`
3. 検索ボックスに **`GitHub Copilot`** と入力
4. **「GitHub Copilot」**（GitHub公式）を見つけて **「Install」** をクリック
   - 同時に **「GitHub Copilot Chat」** も自動的にインストールされる
5. インストール後、VS Code の右下に通知が表示される

### GitHub にサインイン

1. インストール後、右下に **「Sign in to GitHub」** の通知が表示される
2. **「Sign in to GitHub」** をクリック
3. ブラウザが開くので、GitHub にログイン
4. **「Authorize Visual Studio Code」** をクリック
5. VS Code に戻ると、Copilot が有効になっている

### 確認ポイント

- [ ] VS Code の右下のステータスバーに Copilot のアイコンが表示されている
- [ ] アイコンに斜線が入っていない（有効状態）

## 3. Copilot の基本的な使い方

### 3-1. コード補完（自動サジェスト）

Copilot の最も基本的な機能。コードを書いていると、自動的に候補が灰色のテキストで表示される。

#### 試してみよう

1. VS Code で新しいファイルを作成：`hello.py`
2. 以下のコメントを入力してEnterを押す：

```python
# 1から10までの合計を計算する関数
```

3. Copilot が関数のコードを灰色で提案してくれる
4. **`Tab`** キーを押すと提案を受け入れる
5. **`Esc`** キーを押すと提案を拒否する

#### コード補完の操作まとめ

| 操作 | キー |
|------|------|
| 提案を受け入れる | `Tab` |
| 提案を拒否する | `Esc` |
| 次の提案を見る | `Alt + ]` |
| 前の提案を見る | `Alt + [` |
| 複数の提案を一覧表示 | `Ctrl + Enter` |

### 3-2. Copilot Chat（チャット機能）

Copilot に日本語で質問したり、コードの説明を求めたりできる。

#### チャットパネルを開く

- サイドバーの **Copilot Chat アイコン**（吹き出しマーク）をクリック
- またはキーボードショートカット：`Ctrl + Alt + I`

#### 便利な使い方

**コードの説明を聞く：**
1. 分からないコードを選択（ドラッグ）
2. 右クリック → **「Copilot」** → **「Explain This」**
3. チャットパネルに日本語で説明が表示される

**コードの修正を依頼する：**
1. 修正したいコードを選択
2. 右クリック → **「Copilot」** → **「Fix This」**

**テストコードの生成：**
1. テストしたい関数を選択
2. 右クリック → **「Copilot」** → **「Generate Tests」**

### 3-3. インラインチャット

エディタ内で直接 Copilot に指示を出せる。

1. コードの任意の場所にカーソルを置く
2. `Ctrl + I` を押す
3. 入力欄が表示されるので、日本語で指示を入力

例：
- `この関数にエラーハンドリングを追加して`
- `このコードをリファクタリングして`
- `コメントを追加して`

## 4. 実習：Copilot を使ってプログラムを作ってみよう

### 課題1：自己紹介プログラム

1. 新しいファイル `self_intro.py` を作成
2. 以下のコメントを入力して、Copilot の提案を活用する：

```python
# 名前、学年、趣味を入力して、自己紹介文を表示するプログラム
```

3. Copilot の提案を `Tab` で受け入れながらプログラムを完成させる
4. ターミナル（`` Ctrl + ` ``）で実行して動作を確認：

```bash
python self_intro.py
```

### 課題2：簡単な計算プログラム

1. 新しいファイル `calculator.py` を作成
2. 以下のコメントを順番に入力して、Copilot に補完させる：

```python
# 2つの数値を受け取り、四則演算（足し算、引き算、掛け算、割り算）の結果を表示する関数

# メイン処理：ユーザーから2つの数値を入力してもらい、計算結果を表示
```

### 課題3：Copilot Chat で質問してみよう

Copilot Chat パネルを開いて、以下のような質問をしてみよう：

- `Pythonでリストの要素を逆順にする方法を教えて`
- `for文とwhile文の違いは何ですか？`
- `README.mdの書き方を教えて`

## 5. Copilot を使うときの注意点

### やるべきこと

- **提案されたコードを理解してから使う** — Copilot は便利だが、提案が常に正しいとは限らない
- **コメントで意図を明確に書く** — 具体的なコメントほど良い提案が得られる
- **提案を修正・改善する** — そのまま使うのではなく、自分の目的に合わせて調整する

### 気をつけること

- **Copilot に頼りすぎない** — 基礎を理解せずに使うと、学習の妨げになる
- **個人情報やパスワードを含むコードを書かない** — Copilot はクラウドで処理される
- **レポートや課題での利用ルールを確認する** — 授業ごとにAIツールの利用方針が異なる場合がある

## よくあるトラブルと対処法

| 症状 | 対処法 |
|------|--------|
| Copilot のアイコンに斜線が入っている | アイコンをクリックして「Enable」を選択 |
| 提案が表示されない | GitHubへのサインインを確認。右下のCopilotアイコンをクリック |
| 「You don't have access」と表示される | GitHub Educationの承認がまだの可能性。承認状況を確認 |
| 日本語で質問しても英語で返ってくる | チャットで「日本語で回答してください」と追加で指示する |
