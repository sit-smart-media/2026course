# 課題解決実習コース — CLAUDE.md

## コース概要
大学2〜3年生向けの課題解決実習（4単位）。木曜日2〜5限。
社会問題・技術課題への取り組みを通じて実践スキルを育成する。

## ディレクトリ構成
```
2026course/
├── README.md              # コース全体の説明・評価方法
├── 1.知識取得.md           # GitHub, ソフトウェア開発基礎, エージェントAI
├── 2.課題の選定と分析.md
├── 3.解決策の提案.md
├── １回目/                # 各回の授業資料
│   ├── copilot-setup.md
│   ├── github-setup.md
│   ├── uv-setup.md
│   └── plan.md
├── 2回目/
└── images/
```

## コース構成（5フェーズ）
1. **知識習得** — GitHub/Copilot, 現代ソフトウェア開発, エージェントAI, 知覚情報処理
2. **課題選定と分析** — 社会問題・技術課題の特定と分析
3. **解決策の提案** — 実現可能性の評価
4. **プロジェクト実施** — 解決策の実装
5. **発表・報告** — プレゼン、学会発表加点あり

## 評価方法（4単位）
| 項目 | 割合 |
|------|------|
| 出席（16回中11回以上必須） | 20% |
| 毎回の進捗・課題（5段階） | 30% |
| プロジェクト成果・プレゼン | 20% |
| 期末レポート（5段階） | 10% |

## 資料作成ガイドライン
- 言語：日本語（説明資料）
- 授業資料は各回のフォルダ（`１回目/`, `2回目/` 等）に配置
- Markdownで作成し、参考サイトは明示する
- 学生向けのセルフスタディ形式を基本とする
