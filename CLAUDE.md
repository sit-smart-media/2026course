# 課題解決実習コース — CLAUDE.md

## コース概要
大学2〜3年生向けの課題解決実習（4単位）。木曜日2〜5限。
社会問題・技術課題への取り組みを通じて実践スキルを育成する。

## ディレクトリ構成
```
2026course/
├── README.md              # コース全体の説明・評価方法
├── 1.知識取得.md           # GitHub, ソフトウェア開発基礎, エージェントAI
├── 2.課題の選定と分析.md
├── 3.解決策の提案.md
├── 1回目/                # 各回の授業資料
├── 2回目/
......
```

## コース構成（5フェーズ）
1. **知識習得** — 開発環境、現代ソフトウェア開発, 知覚情報処理
2. **課題選定と分析** — 社会問題・技術課題の特定と分析
3. **解決策の提案** — 実現可能性の評価
4. **プロジェクト実施** — 解決策の実装
5. **発表・報告** — プレゼン、学会発表加点あり

## 資料作成ガイドライン
- 授業資料は各回のフォルダ（`１回目/`, `2回目/` 等）に配置、毎回の内容は独立。
- 各回は知識と実践に組み合わせた内容で、前半は知識講義、後半は実践の基本
- 前半の知識講義：日本語（説明資料）のMarkdownで作成し、参考サイトは明示する
- 後半の実践：教師が基本を説明した上で、学生はPCを使って実装を試す。
  - 開発環境：uv tool, github, github desktop, vscode, github copilot
  - 開発言語：基本はpython、必要に応じて、javascript (react, typescript), html/css
- 学生向けのセルフスタディにふさわしい内容
- 本コースは90分は講義と実践の基本を確認してから、さらに、セルフスタディの時間でプラス90分×2を確保して、実践を中心に知識の理解を深める
