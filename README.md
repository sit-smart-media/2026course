
# 課題解決実習コース

本コースは大学2～3年生の課題解決実習の内容をまとめたものです。
課題解決実習は、学生が実際の社会問題や技術的な課題に取り組むことで、
実践的なスキルを身につけることを目的としています。

このコースでは、以下のような内容を扱います。

1. 知識の習得
   - 課題解決に必要な基礎知識や技術を学びます。
   - 例えば、データ分析、プロジェクト管理、2Dから３D構築技術などが含まれます。

2. 課題の選定と分析
   - 社会問題や技術的な課題を選び、その背景や影響を分析します。
   - 
3. 解決策の提案
   - 課題に対する具体的な解決策を提案し、その実現可能性を評価します。
   - 
4. プロジェクトの実施
   - 提案した解決策を実際に実施し、その成果を評価します。
   - 
5. プレゼンテーションと報告
   - プロジェクトの成果をプレゼンテーションや報告書としてまとめ、共有します。
   - チームでの協力やコミュニケーションの重要性も学びます。
   - 

このコースを通じて、学生は問題解決能力やプロジェクト管理のスキルを向上させることができます。また、実際の課題に取り組むことで、社会に対する理解を深めることもできます。

# 時間配分
- 木曜日、2〜５限
  - 　午前：　基本知識の講義と実践（２限〜３限）
  
  - 　午後：　専門知識とプロジェクト実習　（3〜４限）
    - 双方向講義：　教師からプロジェクトの専門知識紹介、各自の自分の調査、進捗発表
      - 発表は評価ポイント
    - ５限は１日の進捗報告（個別面談）、面談後の時間は自由

# 評価方法 (4単位)

- 出席回数
  - 16回のうち11回以上出席が必要、成績に２０％
- 毎回の進捗状況や課題提出の内容に基づいて評価されます。
  - チームワーク、チームメンバーとの協力やコミュニケーションの質も評価されます。
  - 毎回の進捗を５段階評価：　成績に３０％
- プロジェクトの進捗や成果に基づいて評価されます。
  - プレゼンテーション、プロジェクトの成果を発表する際の内容やプレゼンテーションの質も評価されます。
  - 学会発表や論文投稿に加点します。
  - 成績に２０％を反映
- 振り返り
  - 期末レポート課題：５段階評価
  - 成績に１０％
