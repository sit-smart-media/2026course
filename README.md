
# 課題解決実習コース

本コースは大学2～3年生の課題解決実習の内容をまとめたものです。
課題解決実習は、学生が実際の社会問題や技術的な課題に取り組むことで、
実践的なスキルを身につけることを目的としています。

このコースでは、以下のような内容を扱います。

1. 知識の習得
   - 課題解決に必要な基礎知識や技術を学びます。
   - 例えば、データ分析、プロジェクト管理、2Dから３D構築技術などが含まれます。

2. 課題の選定と分析
   - 社会問題や技術的な課題を選び、その背景や影響を分析します。
   - 
3. 解決策の提案
   - 課題に対する具体的な解決策を提案し、その実現可能性を評価します。
   - 
4. プロジェクトの実施
   - 提案した解決策を実際に実施し、その成果を評価します。
   - 
5. プレゼンテーションと報告
   - プロジェクトの成果をプレゼンテーションや報告書としてまとめ、共有します。
   - チームでの協力やコミュニケーションの重要性も学びます。
   - 

このコースを通じて、学生は問題解決能力やプロジェクト管理のスキルを向上させることができます。また、実際の課題に取り組むことで、社会に対する理解を深めることもできます。

# 時間配分
- 木曜日、1〜4/2~5限
  - 　基本知識の理解と実践
  - 　専門知識とプロジェクト実習
    - 双方向：　教師から基本知識や専門知識の講義、各自で調査、自習、実装、進捗発表

# このレポジトリを取得するgitコマンド

取得する：（自分のPCで資料を置きたいフォルダで下記のコマンドを実行する）
```
git clone --depth 1 
```

取得済の場合、最新の資料で更新する：（コースフォルダの下に下記のコマンドを実行する）
```
git pull --depth 1
```
