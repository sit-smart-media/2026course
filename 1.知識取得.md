# GitHubの使用
-　github アカウント作成
  - 学生アカウントを作成して、github copilotの利用を開始します。
  - github copilotは、コードの自動生成や補完を行うAIツールで、ソフトウェア開発の効率を向上させることができます。
  - 参考サイト：
    - https://docs.github.com/ja/copilot/get-started/quickstart
  
- githubの基本的な使い方を学びます。
  - リポジトリの作成、クローン、コミット、プッシュなどの基本的な操作を理解します。
  - チームでの共同作業やコードレビューの方法も学びます。
  - github desktopやコマンドラインツールを使用して、効率的なワークフローを構築します。
  - 参考サイト：
    - https://docs.github.com/ja/desktop/installing-and-authenticating-to-github-desktop/installing-github-desktop
